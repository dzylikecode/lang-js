## 建立目录

对于 js 的教程, 按照自己的结构和逻辑来

比如一个是关于类的, 一个是 c 的结构的

注意沉浸式, 同时需要花一点时间总结

让自己快速地了解, 然后应用, 等需要深入的时候再深入
